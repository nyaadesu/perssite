hi!! this is my first post on my new personal site, ariel.hart.cat~

I finally made myself a little corner of the internet. No algorithm, no feed, no engagement metrics. Just me, a text editor, and some markdown files.

## what this place is

This is where I'll write about things I find interesting. Probably a mix of:

- networking stuff and niche tech I've been digging into
- anime I'm watching
- random thoughts that don't fit anywhere else

No posting schedule, no theme, no pressure. Just vibes.

## how it's made

The site runs on [Eleventy](https://www.11ty.dev/) and is styled to look like Windows 98, because I think that's cute and funny. Writing a new post is just dropping a markdown file into a folder, which is exactly the kind of simplicity I wanted.

Hosted on Nekoweb, part of the [nekonet](https://nekoweb.org/) — a small web revival project I think is really lovely.

Anyway, welcome! Make yourself at home. (｡♥‿♥｡)
